# Axiom Visual Assets

Save the Discord screenshots and demo GIF in this folder using these filenames:

- `pingbomb-session.png` - `/pingbomb` launch, controls, ping output, and completion embed
- `stats-dashboard.png` - `/stats` analytics embed
- `server-settings.png` - `/settings` configuration embed
- `axiom-demo.gif` - optional short workflow demo for the top of the README

These paths are referenced by the repository README.
